# 시공간 웜홀의 입장

![](https://astrokings.s3.ap-northeast-2.amazonaws.com/html/img/help/2102_01.jpg)

시공간 웜홀은 주 1회 입장 가능하며, 매주 수요일 00시(UTC)에 입장 횟수가 초기화됩니다.
또한, 이미 진행 중인 웜홀에도 난입하여 참여할 수 있습니다.

<br>
<br>
<br>

## 시공간 웜홀의 입장 조건

- 우주에 출격 중인 함대가 없어야 함
- 우주에 수리 중인 기함/함선이 없어야 함
- 우주에 파손 상태인 기함/함선이 없어야 함



<br>
<br>
<br>


## 시공간 웜홀의 종류

시공간 웜홀은 사령부 레벨에 따라 별도의 웜홀로 이동합니다.<br>

시공간 웜홀 로우 : 사령부 35레벨 ~ 사령부 43레벨<br>
시공간 웜홀 미들 : 사령부 44레벨 ~ 사령부 50(CL.4)레벨<br>
시공간 웜홀 하이 : 사령부 50(CL.5)레벨 ~ 사령부 50(CL.15)레벨

<br>
<br>
<br>

## 시공간 웜홀 입장 시, 주의 사항

시공간 웜홀에서는 인벤토리를 사용할 수 없습니다.<br>
시공간 웜홀에서 사용하는 가속, 행동력, 크리스탈의 경우 입장 전, 미리 확인해주시길 바랍니다.<br>
