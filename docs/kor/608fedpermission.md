# 연맹 등급별 권한

G5는 연맹의 연맹 의장이자 최고 관리자입니다.
<br>
G4는 연맹의 실질적인 관리자이며 G3~G1은 일반 연맹원입니다.
<br>
연맹 가입 시 기본 등급은 G1입니다.
