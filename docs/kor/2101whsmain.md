# 시공간 웜홀 탐사
![](https://astrokings.s3.ap-northeast-2.amazonaws.com/html/img/help/2101_01.jpg)

***최근 테트메스 은하계에서 발생하는 시공간 웜홀을 상인조합(E.M.U)에서 우선적으로 탐사하였습니다.***

***그러나 현재 상인조합(E.M.U)의 역량만으로는 한계에 부딪혀, 사령관님의 도움이 절실히 필요합니다.***

***시공간 웜홀을 탐사하고 사령관님의 명성을 떨쳐보십시오!***

## 시공간 웜홀의 시기와 진행 

시공간 웜홀은 1주에 두 번 진행됩니다.
- 1회차 : UTC 기준, 금요일 09시
- 2회차 : UTC 기준, 금요일 19시
