# 보상

## 랭킹 보상

![](https://astrokings.s3.ap-northeast-2.amazonaws.com/html/img/help/2104_01.jpg)

각 시공간 웜홀에 따라 랭킹에 시공간 웜홀 보상 토큰이 지급됩니다.

| 랭킹 순위 | 시공간 웜홀 로우 | 시공간 웜홀 미들 | 시공간 웜홀 하이 |
| - | :-: | :-: | :-: |
| 1위 | 3300 | 6300 | 9300 |
| 2위 | 2700 | 5700 | 8700 |
| 3위 | 2200 | 5200 | 8200 |
| 4위 | 1800 | 4800 | 7800 |
| 5위 | 1500 | 4500 | 7500 |
| 6위~10위 | 1200 | 4200 | 7200 |
| 11위~25위 | 1000 | 4000 | 7000 |
| 상위 20% | 8000 | 3800 | 6800 |
| 상위 50% | 700 | 3700 | 6700 |
| 상위 100% | 600 | 3600 | 6600 |

## 시공간 웜홀 상점

![](https://astrokings.s3.ap-northeast-2.amazonaws.com/html/img/help/2104_02.jpg)

랭킹 보상으로 획득한 시공간 웜홀 보상 토큰을 사용하여 허브 강화 재료 및 다양한 아이템으로 교환할 수 있습니다.
