# 1.명성

 적 행성을 공격하거나, 아스트로네스트를 공격하여 승리하면 전쟁 명성이 상승합니다.

같은 상대와의 전투는 일 3회까지만 명성 변동이 인정되며, 그 이상의 반복 전투는 명성에 영향을 주지 않습니다.

상대와의 국력 차이에 따른 명성 변동 규칙은 다음과 같습니다.

| PVP 결과 | 상대의 국력 비율 | 명성 획득 비율 | 명성 변화량 |
| -------- | ---------------- | -------------- | ----------- |
| 승리     | 100% ~           | 150%           | + 30        |
|          | 85% ~ 100%       | 100%           | + 20        |
|          | 50% ~ 85%        | 50%            | + 10        |
|          | 25% ~ 50%        | 25%            | + 5         |
|          | ~ 25%            | 10%            | + 2         |
| 패배     | 300% ~           | 5%             | - 1         |
|          | 200% ~ 300%      | 10%            | - 2         |
|          | 150% ~ 200%      | 25%            | - 5         |
|          | 100% ~ 150%      | 50%            | - 10        |
|          | 75% ~ 100%       | 100%           | - 20        |
|          | ~ 75%            | 150%           | - 30        |
