# 名声

 敵の惑星を攻撃するか、アストロネストを攻撃して勝利すると、戦争名声が上昇します。

同じ相手との戦闘は1日3回まで名声の変動が認められ、それ以上繰り返される戦闘は名声に影響しません。

相手との艦隊戦闘力の差による名声変動規則は次の通りです。

| PVP 結果 | 相手の艦隊戦闘力比率 | 名声獲得比率 | 名声変化量 |
| -------- | ---------------- | -------------- | ----------- |
| 勝利     | 100% ~           | 150%           | + 30        |
|          | 85% ~ 100%       | 100%           | + 20        |
|          | 50% ~ 85%        | 50%            | + 10        |
|          | 25% ~ 50%        | 25%            | + 5         |
|          | ~ 25%            | 10%            | + 2         |
| 敗北     | 300% ~           | 5%             | - 1         |
|          | 200% ~ 300%      | 10%            | - 2         |
|          | 150% ~ 200%      | 25%            | - 5         |
|          | 100% ~ 150%      | 50%            | - 10        |
|          | 75% ~ 100%       | 100%           | - 20        |
|          | ~ 75%            | 150%           | - 30        |
