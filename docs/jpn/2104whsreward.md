# 報酬

## ランキング報酬

![](https://astrokings.s3.ap-northeast-2.amazonaws.com/html/img/help/2104_01.jpg)

各時空間ワームホールのランキングに応じて時空間ワームホール探査トークンが支給されます。

| ランキング順位 | 時空間ワームホール【ロー】 | 時空間ワームホール【ミドル】 | 時空間ワームホール【ハイ】 |
| - | :-: | :-: | :-: |
| 1位 | 3300 | 6300 | 9300 |
| 2位 | 2700 | 5700 | 8700 |
| 3位 | 2200 | 5200 | 8200 |
| 4位 | 1800 | 4800 | 7800 |
| 5位 | 1500 | 4500 | 7500 |
| 6位~10位 | 1200 | 4200 | 7200 |
| 11位~25位 | 1000 | 4000 | 7000 |
| 上位20% | 8000 | 3800 | 6800 |
| 上位50% | 700 | 3700 | 6700 |
| 上位100% | 600 | 3600 | 6600 |

## 時空間ワームホールショップ

![](https://astrokings.s3.ap-northeast-2.amazonaws.com/html/img/help/2104_02.jpg)

ランキング報酬として獲得した時空間ワームホール報酬トークンは、ハブ強化材料や様々なアイテムと交換することができます。
